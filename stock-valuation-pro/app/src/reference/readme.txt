A comprehensive stock valuation plugin with DCF analysis, technical analysis, relative valuation, and news feeds.

== Description ==

**Stock Valuation Pro** brings professional-grade stock analysis tools to your WordPress website. Whether you're running a financial blog, investment education site, or portfolio tracking platform, this plugin provides all the tools your visitors need to analyze stocks.

= Features =

* **DCF Valuation** - Discounted Cash Flow analysis with adjustable WACC, growth rate, and terminal growth assumptions
* **Relative Valuation** - Compare stocks with industry peers using P/E, P/B, EV/EBITDA ratios
* **Technical Analysis** - Interactive price charts with RSI, moving averages, support/resistance levels, and pivot points
* **News Feed** - Latest news articles for any stock ticker
* **Multiple Valuation Methods** - Graham Number, Peter Lynch, EPV, DDM, and Reverse DCF
* **Beautiful Dark Theme** - Modern glassmorphism design that looks stunning
* **Fully Responsive** - Works perfectly on all devices
* **Admin Settings** - Enable/disable specific features as needed
* **Shortcode Support** - Easy embedding on any page or post

= Available Shortcodes =

* `[stock_dashboard]` - Complete analysis dashboard with all features
* `[stock_valuation ticker="AAPL"]` - DCF valuation widget
* `[stock_technicals ticker="AAPL"]` - Technical analysis with charts
* `[stock_relative ticker="AAPL"]` - Peer comparison metrics
* `[stock_news ticker="AAPL"]` - Stock news feed

== Installation ==

1. Upload the `stock-valuation-pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Stock Valuation > Settings to configure your preferences
4. Add shortcodes to any page or post

== Frequently Asked Questions ==

= Do I need an API key? =

The plugin uses Yahoo Finance for basic data, which doesn't require an API key. For enhanced features, you can optionally add a Financial Modeling Prep API key.

= Can I customize the appearance? =

Yes! You can switch between dark and light themes in the settings, and customize the accent color.

= Is the data real-time? =

Stock data is cached for 1 hour by default to optimize performance. News data is cached for 30 minutes.

= Which stock exchanges are supported? =

The plugin supports all major exchanges including NYSE, NASDAQ, LSE, TSX, and more through Yahoo Finance.
